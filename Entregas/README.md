# Entregas

Esta sección documenta las actividades evaluables del curso: talleres, tareas, reportes, quices computacionales o avances de proyecto.

## Convenciones sugeridas

- Publicar instrucciones en Markdown o Notebooks sin soluciones.
- Indicar fecha de entrega, formato esperado y criterios de evaluación.
- Separar datos de entrada, resultados generados y cuadernos de trabajo.
- Pedir que los Notebooks se entreguen limpios, ejecutables y con conclusiones escritas.

## Plantilla de actividad

- **Objetivo:** que debe demostrar el estudiante.
- **Instrucciones:** pasos y restricciones.
- **Datos o insumos:** archivos, enlaces o parámetros.
- **Producto esperado:** Notebook, informe, figura, tabla o repositorio.
- **Rúbrica:** criterios y ponderaciones.
