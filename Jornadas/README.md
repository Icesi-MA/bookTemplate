# Jornadas

Esta sección organiza el trabajo del curso por sesiones, semanas o temas. Cada jornada puede combinar explicación conceptual, exploración computacional, ejercicios guiados y actividades de cierre.

## Formato recomendado para cada jornada

- Objetivos de aprendizaje.
- Conceptos previos.
- Desarrollo matemático.
- Implementación en Python.
- Ejercicios de práctica.
- Preguntas de reflexión o cierre.

Para crear una nueva jornada, duplica `plantillas/cuaderno_base.ipynb`, renombra el archivo dentro de `Notebooks/` y agrega la ruta en `myst.yml`.
