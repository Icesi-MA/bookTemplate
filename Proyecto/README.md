# Proyecto Aplicado

El proyecto aplicado permite conectar los métodos del curso con un problema realista de modelación, análisis de datos, simulación u optimización.

## Estructura sugerida

- **Pregunta o problema:** describe el fenómeno que se quiere estudiar.
- **Modelo matemático:** define variables, supuestos y relaciones.
- **Implementación computacional:** presenta el código usado para explorar o resolver el problema.
- **Análisis de resultados:** interpreta los resultados en el contexto original.
- **Conclusiones:** resume hallazgos, limitaciones y posibles extensiones.

## Entregables sugeridos

- Propuesta breve.
- Avance con modelo y datos.
- Notebook reproducible.
- Presentación final.
- Informe corto o página dentro del Jupyter Book.
